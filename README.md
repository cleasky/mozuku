:whale: One of seaweed fork, client of [rinsuki/sea](https://github.com/rinsuki/sea).

## Getting Started

```sh
# Install dependencies
$ yarn install

# Copy .env.example to .env
$ cp .env.example .env

# Add client_id and client_secret
$ editor .env

# Let's start
$ yarn start

# Remove .cache directory if an error occurs
$ rm -rf .cache
```
